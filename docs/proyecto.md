# Proyecto: Aplicación Web de Gestión de Reservas para Gimnasio PowerGym

## Descripción general
Este proyecto consiste en el desarrollo de una **aplicación web para la gestión de reservas de clases en un gimnasio**.  
El objetivo principal es digitalizar y automatizar la organización de clases, permitiendo a los usuarios consultar actividades, registrarse y reservar plazas, mientras el administrador gestiona horarios, aforos y usuarios.

El contenido del proyecto está basado en la propuesta inicial incluida en la documentación del módulo.

---

## Objetivos del proyecto

### Objetivos generales
- Desarrollar una plataforma web moderna y funcional.  
- Permitir a los usuarios registrarse, iniciar sesión y realizar reservas.  
- Proporcionar al administrador una interfaz para gestionar clases, horarios y usuarios.

### Objetivos específicos
- Implementar autenticación y control de roles (usuario/administrador).  
- Diseñar e implementar una interfaz responsive basada en prototipos de Figma.  
- Construir un backend con Node.js y Express conectado a MySQL.  
- Validar formularios y controlar errores en cliente y servidor.  
- Mantener una estructura ordenada con buenas prácticas de desarrollo web.

---

## Funcionalidades principales

### Para usuarios
- Registro e inicio de sesión.  
- Visualización de horarios y clases disponibles.  
- Reserva de clases y gestión de sus propias reservas.

### Para administradores
- Creación, edición y eliminación de clases.  
- Gestión de usuarios.  
- Control del aforo y visualización del estado de las reservas.

---

## Tecnologías utilizadas
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** Node.js + Express.js  
- **Base de datos:** MySQL  
- **Diseño UI:** Figma  
- **Entorno de desarrollo:** Visual Studio Code  
- **Control de versiones:** Git + GitHub  

---

## Fases del proyecto
1. **Análisis y diseño** (requisitos, BD, prototipos)
